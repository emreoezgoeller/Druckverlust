## Phase 18.27
### Phase 18.27 – Teilstrecken-Schnellerfassung

Die Anlagenübersicht enthält neu eine kompakte Schnellerfassung für Teilstrecken. Luftmenge, Länge, Typ und Geometrie können direkt in der Übersicht angepasst werden. Zugeordnete Formteile werden automatisch mit der Teilstrecke synchronisiert, sofern kein manueller Override gesetzt wurde.


- Verdichtete Ergebnisdetails in Anlagen-, Teilstrecken- und Formteilansicht.
- Kritische Teilstrecke wird hervorgehoben.
- Formteil-Nachweis zeigt p_dyn, ζ und Δp verständlicher.
