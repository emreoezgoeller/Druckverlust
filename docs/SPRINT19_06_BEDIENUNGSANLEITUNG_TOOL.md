# Phase 19.06 – Bedienungsanleitung im Tool

## Ziel

Die Produktseite bleibt kurz. Die ausführlichere Bedienungsanleitung sitzt direkt im Tool unter **Hilfe**.

## Umsetzung

- Neue Hilfe-Auswahl `help` im Workspace.
- Ribbon-Button **Hilfe** öffnet die Hilfeseite im Arbeitsbereich.
- Direktlink von der Produktseite über `app.html?help=1`.
- Hilfeseite mit Schnellstart, Eingabeübersicht, QS-Hinweisen und Kurzbefehlen.
- Direkte Aktionsbuttons zu Projektangaben, Anlagenübersicht, Formteil-Assistent, Bericht und Demo.

## Nutzen

Neue Nutzer verstehen den Ablauf schneller, ohne dass die Hauptseite mit Text überladen wird.
