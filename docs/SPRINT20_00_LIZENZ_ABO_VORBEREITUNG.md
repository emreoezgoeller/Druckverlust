# Phase 20.00 – Lizenz-/Abo-Vorbereitung

## Ziel

Die öffentliche Produktstruktur wird für spätere Lizenzierung vorbereitet, ohne bereits Zahlung, Login oder Zugriffssperren zu aktivieren.

## Umgesetzt

- Produktseite um kompakten Bereich **Lizenz / Ausblick** erweitert.
- Neue Seite `lizenz.html` mit vorbereiteten Nutzungsstufen erstellt.
- Footer und Navigation um Lizenzlink ergänzt.
- `site.webmanifest` und `sitemap.xml` um Lizenzübersicht ergänzt.
- Neue technische Konfigurationsdatei `src/licensing/licenseConfig.js` ergänzt.
- Cache-Version auf `20.00` erhöht.

## Wichtig

Phase 20.00 ist nur eine Vorbereitung. Es gibt noch keine Zahlung, keinen Login, keine technische Sperre und keine verbindlichen Abo-Bedingungen.
