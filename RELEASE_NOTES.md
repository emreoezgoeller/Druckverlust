# Druckverlust Pro 2.0 – Release Notes

**Release:** 2.0.0 · Phase 45.00  
**Datum:** 19.07.2026

## Inhalt

- vollständige Zusammenführung der Entwicklungsphasen 21.12 bis 42.00,
- bereinigte Projektstruktur mit stabilen Produkt- und Feedback-Assets,
- aktuelle Startseite, Qualitätshinweise, Manifest und Dokumentation,
- zusätzliche Rückwärtskompatibilitäts- und Lastprüfungen bis 200 Teilstrecken,
- erneute Prüfung von Berechnung, 14 Formteiltypen, Excel-Referenzen, Speicherung, Mehranlagen, Schema, Simulation und Professional Report,
- Windows-Starter für den lokalen Betrieb über `http://`,
- finale Cache-Kennung 45.00 für die integrierte Release-Version.

## Bewusst ausgeschlossen

- keine Ventilatorauslegung, Motorleistung, SFP-, Energie- oder Betriebskostenberechnung,
- keine Hersteller-, Produkt- oder Artikelnummerndatenbank.

## Installation / Start

1. ZIP vollständig entpacken.
2. Unter Windows `Druckverlust_starten.bat` ausführen.
3. Auf der Produktseite **Tool starten** wählen.

`app.html` nicht direkt über `file://` öffnen.
